v1.0.0

Version v1.0.0 will contain the basic information needed for players for the upcoming campaign playing in this world. Namely, v1.0.0 will have information regarding Hasanko's Archipelago (the starting islands), the lore (every page in that category), the maps (world and Hasanko's Archipelago) and the reference page. In fact, players should have enough information to create characters at this stage of the website (although the DM will not have the information to run a campaign - see v2.0.0).

At the moment, there are 4 items on my checklist that still need to be done before this the full release of v1.0.0 occurs.
